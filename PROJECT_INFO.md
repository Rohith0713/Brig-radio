# Project Information: CampusWave

## Tech Stack

### Backend
- **Core:** Python 3.x, Flask
- **Database:** SQLAlchemy (ORM), MySQL (PyMySQL)
- **Authentication:** Flask-JWT-Extended
- **Environment:** python-dotenv
- **Integrations:** Firebase Admin SDK
- **Production Support:** Gunicorn, Procfile (Heroku ready)

### Android Frontend
- **Languages:** Kotlin
- **UI Frameworks:** Jetpack Compose, MD3
- **Networking:** Retrofit, OkHttp, Gson
- **Development Patterns:** MVVM, Coroutines, Flow
- **Data:** DataStore (Preferences), SQL (implicit through Room or just Retrofit)
- **Media Support:** Media3 (ExoPlayer), Coil (Image loading)
- **Features:** WebRTC (Signaling), Firebase Messaging
